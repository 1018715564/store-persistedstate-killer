# store-persistedstate-killer

[EN](https://github.com/1018715564/store-persistedstate-killer/README.md) / 中文

🥷 杀手级别的持久化状态管理库

- [x] 可以为多个库提供持久化服务 (vuex, pinia)
- [x] 支持 TypeScript
- [x] 支持 LocalStorage
- [x] 支持相对安全的存储环境（非明文）
- [x] 自由的安装方式
- [x] 灵活的配置且没有副作用
- [x] 对开发友好的状态变更 Log
- [x] 持久化加强功能 (过期时间,重命名...)
- [x] 也提供了开箱即用的加密服务（简单但是好用）

## 安装

```
npm i store-persistedstate-killer
```
